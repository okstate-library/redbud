Inventory Loans

Inventory Loans Report- Help

This report includes two dropdowns—one for location (Oklahoma State University Creative Studio and Laptop) and another for year. Using the selected location and year, the system accesses the circulation loans API to collect open loan records. For each record, the user API is called using the user’s barcode to retrieve user details. The report displays the loan type, user information (primary identifier and name), item policy, due date, and barcode.
API’s
  • /circulation/loans?query=(status.name==’open’ and itemEffectiveLocationIdAtCheckOut==’location’ AND dueDate >= ‘year’-01-01' AND dueDate < 'year’-12-31')
  • /users?query=(barcode==barCode)
