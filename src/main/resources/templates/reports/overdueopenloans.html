Overdue Open Loans

Overdue Open Loans Report - Help

Collects open loans based on selected locations using the API. The user must first choose a location to proceed. For each open loan item, an additional API call retrieves details such as the barcode and title. User information is then obtained through the user details API based on the borrower associated with the open loan.
API’s
  • /loan-storage/loans?query=(status.name==’open’ AND checkoutServicePointId==‘servicePoint’)
  • /inventory/items/itemId
  • /users?query=(id==userId)
