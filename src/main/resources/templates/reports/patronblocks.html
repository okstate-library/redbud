Patron blocks Report - Help

First, the manual block API is used to collect all manual blocks. These are then filtered by the institution ID captured from the user interface. While iterating through the blocks, the system retrieves user details via the user API. The report includes user information (primary identifier, name, and email), along with the block type and code. It also indicates whether the user can borrow, renew, or place requests. When a report row is expanded, the block description is displayed.
